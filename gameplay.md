# Gameplay
Warmancer Black Friday is a turn-based strategy board game based around the concept of imperfect information where two players compete to dominate the battlefield with their hidden pawns and destroy each other's thrones.

## Game Flow
The gameplay is structured into distinct phases: Pre-Game, Setup Phase, Move Phase, Resolve Phase, and End Phase on the client side. In the context of this section, the word Server is interoperable with Smart Contract, because the API is designed to work the same for both.
